  1. Organização do Site Pessoal

    Tarefa: Hora de colocar em ordem no seu repositório GitHub Pages.

    • Disponibilizar uma página index.html bem formatada, com links para as atividades que serão solicitadas ao longo do curso.

    • Não pode haver links quebrados ou páginas que não permitam navegar para as demais páginas.
    • Seu site deve obedecer a um padrão de diagramação e cores.
