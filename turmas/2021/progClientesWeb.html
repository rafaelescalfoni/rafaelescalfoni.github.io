Laboratório - Aplicativo Web

Nesta atividade iniciada em sala de aula dedica-se a construção de uma aplicação web com funcionalidades cliend-side. Baseado na atividade do curso HTML, CSS e JavaScript para desenvolvedores do Coursera/Johns Hopkins, complete a atividade com as duas atividades propostas no arquivo script.js.

Seu objetivo é completar a implementação de um menu de um restaurante chinês. O app é single page application SPA, ou seja, todas as funcionalidades são renderizadas na mesma tela. A tela inicial, a apresentação dos tipos de pratos e o detalhamento de pratos já estão funcionando. Você deve completar a construção da funcionalidade de pedido.
