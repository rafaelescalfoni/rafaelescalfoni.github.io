Aulas

Materiais de apoio

  • Aula1

    Introdução a Programação Client side

    Discutindo a evolução da internet, tecnologias relacionadas e a guerra dos navegadores.

  • Aula2

    Rodando Javascript

    Apresentando maneiras de executar seu código Javascript. Monte sua caixa de ferramentas de desenvolvimento JS.

  • Aula3

    Linguagem Javascript

    Conceitos básicos a respeito da sintaxe e estrutura da linguagem.

  • Aula4

    Estruturas de Repetição

    Conheça as estruturas de iteração em Javascript.

  • Aula5

    Tipos de dados

    Conheça os tipos primitivos da linguagem Javascript. Conheça algumas peculiaridades da linguagem.

  • Aula6

    Arrays e Objetos

    Descubra estruturas mais complexas em Javascript.

  • Aula7

    Visibilidade de variáveis

    Entenda como o escopo das variáveis é definida em Javascript.

  • Aula8

    Funções

    Funções de primeira classe? Hoisting? Callback? Lambda? Melhor não perder essa aula ;-)

  • Aula9

    String API

    Conhecendo os métodos do objeto String

  • Aula10

    Date API

    Conhecendo os métodos do objeto Date.

  • Aula11

    jQuery

    Conceitos básicos, seletores. Efeitos e manipulação do DOM. Eventos.

  • Aula12

    Manipulando o DOM

    Manipulando páginas web com Javascript.

  • Aula13

    Requisições HTTP e jQuery AJAX

    Como funciona o HTTP, seus métodos, código de status, cabeçalhos e formatos. Carregando páginas dinamicamente com o jQuery.

  • Aula14

    jQuery e plugins

    O que é um plugin? Escrevendo plugins.

  • Aula15

    API Fetch

    Conheça a interface JavaScript para acesso e manipulação de partes do pipeline HTTP.

  • Aula16

    Armazenamento - Web Storage

    Session Storage e Local Storage.

  • Aula17

    ECMAScript 6

    Compatibilidade e o Babel.

  • Aula18

    ECMAScript 6 e a componentização

    Modularização e programação orientada a objetos.
